## 📘 프로젝트 소개

**덕스토랑(DuckStaurant)**은 덕성여자대학교 학생들을 위한 학식 기반 모바일 플랫폼으로,  
학생들이 **더 편리하고 재미있게 학식을 이용할 수 있도록** 설계된 서비스입니다.

기존의 단순 학식 제공 앱과 달리 덕스토랑은 다음과 같은 차별점을 지닙니다:

- **하이퍼리얼 학식 메뉴 구현**
- **헬스(칼로리) 기반 메뉴 추천 시스템**
- **친구에게 메뉴를 선물하는 기능**
- **가게(사장님)용 관리자 페이지 제공**

학생 사용자와 가게 관리자가 동시에 사용할 수 있는 **실서비스 형태의 학식 앱**입니다.

---

## 🎯 기획 의도


프로젝트는 "덕성여대만의 학식 애플리케이션을 만들어보자"는 주제에서 출발했습니다.

대부분의 학생이 매일 이용하는 학식 서비스지만, 기존 학식 앱들은 단순히 메뉴를 나열하는 방식에 그쳐 사용성이 제한적이었습니다.  
특히 활동량 기반 식단 추천이나 학식 선물하기 같은 유용한 기능이 아이디어로 떠올랐습니다.

이에 우리는 다음과 같은 목표를 설정했습니다:

- 사용자의 활동량(칼로리)과 식단을 연계한 추천 기능
- 학식을 선물하는 새로운 상호작용 요소
- 학생과 가게 관리자가 함께 사용할 수 있는 실제 서비스 구조

## 🧭 기능 소개

### 🔹 학식 주문

- 실제 학식당 메뉴 데이터 기반 UI
- 식당별 메뉴 카테고리화 및 직관적인 탐색

### 🔹 칼로리 기반 추천

- 사용자의 소모 칼로리를 분석하여 맞춤 메뉴 추천
- 개인 맞춤형 추천 알고리즘 적용

### 🔹 선물하기

- 친구에게 학식 선물 기능
- 선물 내역(받은/보낸) 조회

### 🔹 공지사항

- 가게 관리자의 공지사항 작성 및 등록
- 식당별 공지사항 열람

### 🔹 가게 관리자

- 사용자 주문 목록 실시간 확인
- 조리 완료 시 사용자에게 알림 전송

### 🔹 리뷰

- 가게별 리뷰 목록 확인
